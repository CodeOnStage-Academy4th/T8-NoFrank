## 📱 소개

> 설명아침마다 알람 종료 버튼을 ‘툭' 끄고 다시 잠드는 당신, 이제 돌을 부숴보세요. CRock은 알람이 울리면 스마트폰 속의 돌을 강하게 흔들어 깨야만 꺼집니다. 이 돌이 깨질 때까지 알람은 멈추지 않죠. 이제, 돌을 깨야 하루가 시작됩니다.

[🔗 앱스토어/웹 링크](https://example.com)
